Project: Class Note



This is a Computer Organization Architecture Project which has been done using Assembly Code and concepts. The functionality of this project is there are some chapters and the chapters contain several topics. Using the "Class Notebook", you'll get to see all the chapters and then you'll be asked to choose one fo the chapter and then the topics under that chapter will be shown and as soon as you select one of the topic, you'll be able to see the definition of that topic. In addition to, on that page you will get 3 options to choose. In the whole process if you press anything unexpected that it will take you to the invalid section and ask for to try again.

1. For going back to the main menu.
2. For going back to that chapters topic page that you have read just now.
3. For exit.
